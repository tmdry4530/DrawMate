## 12. E2E 시나리오

### 시나리오 A: 포트폴리오 등록 → 홈 노출 → 검색

```
1. /studio에서 "새 포트폴리오" 클릭
2. 템플릿 선택 (미니멀 그리드)
3. 이미지 2장 업로드 + 커버 지정
4. 기본 정보 입력 (제목, 요약)
5. 분야 태그 1개 + 스킬 태그 2개 선택
6. "게시하기" → 성공 toast → /studio 이동
7. 홈(/)에서 방금 등록한 포트폴리오 카드 확인
8. /explore에서 선택한 분야 태그로 필터링 → 내 포트폴리오 노출 확인
9. 포트폴리오 상세 클릭 → 이미지/태그/가격 정상 표시 확인
```

- [ ] 전체 플로우 막힘 없이 완료
- [ ] 게시 후 홈/탐색에 즉시 노출

### 시나리오 B: 포트폴리오 열람 → 북마크 → 메시지

```
1. /explore에서 타인 포트폴리오 클릭
2. 포트폴리오 상세 열람 (이미지/태그/가격 확인)
3. 북마크 추가
4. "메시지 보내기" 클릭 → 대화방 이동
5. 텍스트 메시지 전송
6. /bookmarks에서 북마크한 포트폴리오 확인
7. /messages에서 대화 목록에 새 대화 표시 확인
```

- [ ] 북마크 상태 일관성 (상세/목록 동기화)

```
1. /studio/portfolios에서 기존 포트폴리오 "편집" 클릭

```
