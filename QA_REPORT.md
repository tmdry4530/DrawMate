# DrawMate QA 체크리스트 테스트 결과

**테스트 날짜:** 2026-03-23
**테스터:** Claude (자동화 QA)
**배포 URL:** https://drawmate.xyz

---

## 핵심 발견 사항 (Critical/Blocking Bugs)

### 🔴 P0 — 서버 크래시: `tagIds: []` 포함 PATCH 요청 시 500

**영향:** 포트폴리오 생성 위저드 전체 차단 (Step 1 → Step 2 진행 불가)
**재현:** 위저드가 자동 저장 시 `tagIds: []`를 포함해 PATCH 호출 → 서버 응답 "태그 동기화에 실패했습니다" (500). `tagIds` 없이 PATCH 하면 200 정상. 빈 배열 핸들링 누락.

### 🔴 P0 — 북마크 API 500 에러

**영향:** POST `/api/v1/portfolios/{id}/bookmark` 가 항상 500을 반환. 북마크 기능 전체 사용 불가.

### 🔴 P0 — 메시지 전송 API 500 에러

**영향:** POST `/api/v1/conversations/{id}/messages` 가 500을 반환. 메시지 전송 불가.

### 🔴 P1 — `/api/v1/me` 엔드포인트 404

**영향:** 설정 페이지 로드 시 "설정 정보를 불러오지 못했습니다" 에러. 프로필 편집 폼에 기존 데이터 미로딩.

### 🟡 P1 — 탐색 필터 버그: 단일 태그 클릭 시 같은 카테고리 전체 선택

**영향:** 분야 탭에서 "웹툰" 하나만 체크하면 5개 분야 태그가 모두 체크됨. 필터링 기능 사실상 무용지물.

### 🟡 P1 — sitemap.xml / robots.txt 도메인 불일치

**영향:** 모든 URL이 `drawmate.kr`로 설정되어 있으나 배포 도메인은 `drawmate.xyz`. SEO 크롤링 불가.

### 🟡 P1 — 공개 프로필 포트폴리오 그리드 미렌더링

**영향:** `/users/[userId]`에서 "2 작품" 표시되나 실제 포트폴리오 카드가 표시되지 않음.

### 🟡 P2 — OG 메타태그 미구현

**영향:** 포트폴리오 상세 페이지에 `og:title`, `og:description`, `og:image` 없음. SNS 공유 시 미리보기 불가.

### 🟡 P2 — `/explore?fieldTags[]=...` URL 파라미터 미반영

**영향:** 홈 인기 태그 클릭으로 explore 이동 시 URL에 필터 파라미터가 있으나 UI 체크박스에 반영 안 됨.
